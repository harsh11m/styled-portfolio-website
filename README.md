# Styled Portfolio Website

This is a personal portfolio website developed using HTML5 and CSS3. The project presents information about my skills, education, and contact details in a structured, clean, and professional format.

---

## Features

- Semantic HTML5 structure (header, main, section, footer)
- External CSS styling using a separate stylesheet
- Responsive design for different screen sizes
- Navigation menu with internal links
- Contact form with input validation
- Integration of Google Fonts
- Basic accessibility features (alt text, labels, ARIA attributes)

---

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
